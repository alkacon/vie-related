Project background

Alkacon started this project motivated by having a playground to evaluate client software components provided by the IKS software stack that can potentially integrated into a future version of OpenCms.

VIE as the client software of IKS is a JavaScript library for implementing decoupled Content Management Systems and semantic interaction in web applications. Alkacon integrated an "image search widget" based on VIE into the TinyMCE editor that is shipped with OpenCms by default.

Current features

  • Getting google flickr images that are related to the written text
  • Getting the depection of dbpedia related entities
  • Integration into TinyMCE
  • Single click search

Upcoming features

  • Getting other content (not only images) related to the text that is stored in OpenCms
  • SEO by using schema.org annotated content for delivered web pages

Getting started

  • Download the projects ZIP-File and open the index.html in the root directory
